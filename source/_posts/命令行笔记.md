### kill process
```javascript
ps aux | grep xx
kill -9 xxx
```
### 输出最后，一般用来看log
```javascript
tail xxx.xx
```
### 输出树形结构
```javascript
tree -L x (x为层级)
```

### 启动服务
```javascript
service xxxx start (xxxx一般指nginx，mongodb等)
```

### ssh免密码登录
添加公匙到`.ssh/authorized_keys`，或`/home/admin/.ssh/authorized_keys`(管理权限)

##### 查看模块具体信息

```
brew info nginx
```

##### 服务列表查看

```
brew services list
```

##### brew命令行启动关闭服务（比如数据库）

```
brew tap homebrew/services
```

##### 安装软件

```
brew tap homebrew/cask-versions
brew cask install xxx
```

##### 删除无用依赖

```
brew autoremove
```

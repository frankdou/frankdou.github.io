##### 查看模块具体信息

```shell
brew info nginx
```

##### 服务列表查看

```shell
brew services list
```

##### brew命令行启动关闭服务（比如数据库）

```shell
brew tap homebrew/services
```

##### 安装软件

```shell
brew tap homebrew/cask-versions
brew cask install xxx
```

##### 删除无用依赖

```shell
brew autoremove
```

1. 需要在~/.ssh文件夹下操作

   ```shell
   t代表type，f代表文件名，C代表注释
   
   ssh-keygen -t rsa -f coding -C ‘coding.net’
   ```

2. 配置~/.ssh/config文件

   ```shell
   ###########
   ###GIHUB###
   ###########
   Host github.com
       HostName github.com
       User xxxx@xxx.com
       IdentityFile ~/.ssh/github
   
   ###########
   ###CODING##
   ###########
   Host git.coding.net
       HostName git.coding.net
       User xxx@xxx.com
       IdentityFile ~/.ssh/coding
   ```

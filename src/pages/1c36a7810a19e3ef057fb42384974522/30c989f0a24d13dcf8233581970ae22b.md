##### 推送单个tag
```shell
git push origin v1.2
```
##### 删除本地所有tags
```shell
git tag | xargs -n1 git tag -d
```
##### 获取远程所有tags
```shell
git fetch --tags
```
##### 删除所线上有tag
```shell
git tag -l | xargs git push origin --delete
```
##### 删除本地所有tag
```shell
git tag -l | xargs git tag -d
```

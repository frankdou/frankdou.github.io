WXML 提供两种文件引用方式`import`和`include`

##### import的作用域

变量需要一层层传递

```vue
<!-- a.wxml -->
<template name="A">
  <text> A template </text>
</template>

<!-- b.wxml -->
<import src="a.wxml"/>
<template name="B">
  <text> B template </text>
</template>

<!-- c.wxml -->
<import src="b.wxml"/>
<template is="A"/>  <!-- Error! Can not use tempalte when not import A. -->
<template is="B"/>
```

```vue
//item.wxml
<template name="item">
  <text>{{text}}</text>
</template>

//index.wxml
<import src="item.wxml"/>
<template is="item" data="{{text: 'forbar'}}"/>
```

##### include

- 可以将目标文件除了 <template/> <wxs/> 外的整个代码引入，相当于是拷贝到 include 位置
- include可以直接使用页面的变量

```vue
<!-- index.wxml -->
<include src="header.wxml"/>
<view> body </view>
<include src="footer.wxml"/>

<!-- header.wxml -->
<view> header </view>

<!-- footer.wxml -->
<view> footer </view>
```

##### setData

###### 进行细粒度更新

```javascript
this.setData({
	[`data[${idx}][${index}].a`]: a,
	[`data[${idx}][${index}].b`]: b,
})
```

###### 数组追加数据

```js
let newData=[{...},{...},{...},{...},{...},{...}];
let changeData={};
let index = this.data.listData.length

newData.forEach((item) => {
	changeData['listData[' + (index++) + ']'] = item //赋值，索引递增
}) 

this.setData(changeData)
```

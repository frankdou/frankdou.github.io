类似vue中的mixins

- 每个组件可以引用多个 behavior ，behavior 也可以引用其它 behavior

- 可绑定页面的生命周期

  - 组件、Behavior可绑定的生命周期一样
  
  ```javascript
  // 页面
  pageLifetimes: {
      show: function() {
        // 页面被展示
      },
      hide: function() {
        // 页面被隐藏
      },
      resize: function(size) {
        // 页面尺寸变化
      }
  }
  // 组件自身
  created: function () {
    console.log('[my-component] created')
  }
  attached: function () { 
  	console.log('[my-component] attached')
  }
  ready: function () {
  	console.log('[my-component] ready')
  }
  ```

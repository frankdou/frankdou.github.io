##### 对象一致性对比

```javascript
JSON.stringify({ a: 1 }) === JSON.stringify({ a: 1 })


// 对象有同一个引用才可以
var foo = { a: 1 };
var bar = { a: 1 };
Object.is(foo, foo);         // true
Object.is(foo, bar);         // false
```

##### 数组一致性对比

```javascript
[1,2].toString() === [1,2].toString()
```

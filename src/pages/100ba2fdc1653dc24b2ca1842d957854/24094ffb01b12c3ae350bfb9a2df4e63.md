#### 闭包

- 函数和对其周围词法环境的引用捆绑在一起构成闭包（实现外部作用域访问内部作用域中变量的方法）

- 高阶函数除了可以接受函数作为参数外，还可以把函数作为结果值返回

- 用途
  - 可以读取函数内部的变量
  - 让变量的值始终保持在内存中

```js
var pet = function(name) {          //外部函数定义了一个变量"name"
  var getName = function() {
    //内部函数可以访问 外部函数定义的"name"
    return name;
  }
  //返回这个内部函数，从而将其暴露在外部函数作用域
  return getName;
};
myPet = pet("Vivie");

myPet();                            // 返回结果 "Vivie"
```

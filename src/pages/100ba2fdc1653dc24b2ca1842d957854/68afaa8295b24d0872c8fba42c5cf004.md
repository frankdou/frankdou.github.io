title: 普通数据作为文件上传
size: 1011
---
```js
const textData = "This is the content of the file.\nIt can contain multiple lines of text.";
const filename = "my_text_file.txt";
const mimeType = "text/plain"; // Set the correct MIME type for the text data

// 2. Create a Blob object from the text data.
const blob = new Blob([textData], { type: mimeType });

// 3. Create a new FormData object.
const formData = new FormData();

// 4. Append the Blob as a file using the three-argument append() method:
//    formData.append(name, blobValue, filename)
formData.append('file_field_name', blob, filename);

// The 'file_field_name' is the name your server-side script will use
// to access the uploaded file (like the 'name' attribute of a file input).

// 5. Send the FormData using AJAX (e.g., with Fetch or XMLHttpRequest):
// Using Fetch API:
fetch('/your-upload-endpoint', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => console.log('Success:', data))
.catch(error => console.error('Error:', error));
```

##### 基础

 `https://www.baidu.com/s?news=1`

- 协议
  - https
- 域名
  - `www.baidu.com`
- path
  - /s
- query
  - ?news=1



##### 基本过程

1. 域名dns解析得到访问ip
2. https协议默认访问443端口
3. 请求方法、路径、请求头
4. 状态码 + 响应头



##### 网站组成

- 域名
- 服务器
- 后端
  - 静态文件
  - 动态内容
    - 数据库
- 前端
  - html、css、js、etc

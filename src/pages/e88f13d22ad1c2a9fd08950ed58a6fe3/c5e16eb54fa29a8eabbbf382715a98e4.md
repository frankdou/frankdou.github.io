- vscode文件跳转支持
- 路径简写

##### webpack配置

```js
resolve: {
	...,
	alias: {
		"@/src": path.resolve(__dirname, "src"),
		...
	},
},
```

##### jsconfig.json
```json
{
  "compilerOptions": {
    "target": "esnext",
    "baseUrl": "./",
    "jsx": "react",
    "paths": {
      "@/src/*": ["src/*"],
      ...
    }
  }
}
```

##### tsconfig.json
```json
{
	"baseUrl": "./",
	"paths": {
		"@/src/*": ["src/*"],
		...
	}
}
```

##### 使用
```jsx
import xxx from "@/src/xxxxx"
```

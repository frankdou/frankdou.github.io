---
##### tsconfig.json

```json
{
    "compilerOptions": {
        "module": "commonjs",
      	// 输出目录
        "outDir": "lib",
      	//自动生成声明文件
        "declaration": true,
       	// 仅生成 .d.ts 文件，不生成 .js 文件
        "emitDeclarationOnly": true,
    }
}
```

```
/path/to/project
├── lib
|  ├── bar
|  |  ├── index.d.ts
|  |  └── index.js
|  ├── index.d.ts
|  └── index.js
├── src
|  ├── bar
|  |  └── index.ts
|  └── index.ts
├── package.json
└── tsconfig.json
```

自动生成的声明文件基本保持了源码的结构，而将具体实现去掉了

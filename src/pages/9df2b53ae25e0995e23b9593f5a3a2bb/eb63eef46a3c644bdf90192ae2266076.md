通过swc-loader，在webpack中使用
```js
 module: {
	 ...
	rules: [
		...
		{
			test: /\.(js|jsx)$/,
			exclude: /node_modules/,
			use: {
				loader: "swc-loader",
				options: {
					"jsc": {
						"parser": {
							"jsx": true,
						}
					},
					"env": {
						"targets": "last 1 chrome version" // 线上ie 11
					}
				}
			}
		}
		...
	]
	...
}
```

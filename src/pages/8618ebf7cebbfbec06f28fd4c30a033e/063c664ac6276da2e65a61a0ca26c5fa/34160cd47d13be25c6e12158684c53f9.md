##### 自定义复杂类型

###### 基本标量类型的合集

```shell
type Product  {
  id: ID,
  name: String,
  price: Float,
  inStock: Int,
  isFreeShipping: Boolean
}
```

###### 枚举

```shell
enum Gender {
	Male
	Female
}

type User {
	id: ID,
	gender: Gender
}
```

###### 列表

```shell
type User {
	id: ID,
	//多个昵称
	nickname: [String],
	favourites: [Product]
}
```

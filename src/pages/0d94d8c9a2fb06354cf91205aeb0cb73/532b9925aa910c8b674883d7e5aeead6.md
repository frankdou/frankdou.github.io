##### 样式

比较：https://github.com/andreipfeiffer/css-in-js/blob/main/README.md

- classNames

  - 基于条件添加class名

- styled-components

  - 动态style标签

  - 支持动态传参、条件判断

  - className通过hash创建，避免命名冲突

    ```jsx
    const Title = styled.h2`
      font-size: 24px;
      color: ${props => props.color};
    `;
    const Component = () => {
      return <Title color="red">Hello World</Title> 
    };
    ```

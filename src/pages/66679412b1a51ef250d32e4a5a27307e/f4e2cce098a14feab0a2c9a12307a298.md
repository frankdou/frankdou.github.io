#### alias

##### webpack配置

```js
alias: {
	// 最外层
	'@/base': path.resolve(__dirname, '..'),
	// utils
	'@/utils': path.resolve(__dirname, '..', 'utils'),
}
```

```jsx
import { data } from '@/base/xxxx'
```

##### tsconfig.json

用来支持vscode快速跳转

```json
{
	"baseUrl": ".",
  "paths": {
    "@/base/*": ["*"],
    "@/utils/*": ["utils/*"]
  }
}
```
